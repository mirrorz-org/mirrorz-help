---
title: GitLab Runner 软件仓库镜像使用帮助
cname: 'gitlab-runner'
---

**注意：gitlab-runner 镜像支持 x86-64 和 ARM64(aarch64) 架构**

### Debian/Ubuntu 用户

首先下载 GitLab 的 GPG 公钥：

<CodeBlock>
```bash
curl -L https://packages.gitlab.com/runner/gitlab-runner/gpgkey | {{sudo}}gpg --dearmor -o /etc/apt/keyrings/gitlab-runner.gpg
```
</CodeBlock>

再选择你的 Debian/Ubuntu 版本，文本框中内容写进 `/etc/apt/sources.list.d/gitlab-runner.list`

<CodeBlock
  enableQuickSetup
  filepath="/etc/apt/sources.list.d/gitlab-runner.list"
  menus={[
    {
      title: '发行版',
      items: [
        ['Debian 13', { os_name: 'debian', release_name: 'trixie' }],
        ['Debian 12', { os_name: 'debian', release_name: 'bookworm' }],
        ['Debian 11', { os_name: 'debian', release_name: 'bullseye' }],
        ['Ubuntu 24.04 LTS', { os_name: 'ubuntu', release_name: 'noble' }],
        ['Ubuntu 22.04 LTS', { os_name: 'ubuntu', release_name: 'jammy' }],
      ]
    }
  ]}
>

```properties
deb [signed-by=/etc/apt/keyrings/gitlab-runner.gpg] {{http_protocol}}{{mirror}}/{{os_name}} {{release_name}} main
```

</CodeBlock>


安装 gitlab-runner:

<CodeBlock>
```bash
{{sudo}}apt-get update
{{sudo}}apt-get install gitlab-runner
```
</CodeBlock>

### CentOS/RHEL

新建 `/etc/yum.repos.d/gitlab-runner.repo`，内容为


<CodeBlock
  enableQuickSetup
  filepath="/etc/yum.repos.d/gitlab-runner.repo"
>

```ini
[gitlab-runner]
name=gitlab-runner
baseurl={{http_protocol}}{{mirror}}/yum/el$releasever-$basearch/
repo_gpgcheck=0
gpgcheck=0
enabled=1
gpgkey=https://packages.gitlab.com/gpg.key
```

</CodeBlock>

再执行

<CodeBlock>
```bash
{{sudo}}yum makecache
{{sudo}}yum install gitlab-runner
```
</CodeBlock>
