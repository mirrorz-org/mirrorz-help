PyPI 镜像在每次同步成功后间隔 5 分钟同步一次。

### pip

#### 临时使用

<CodeBlock>

```
pip install -i {{http_protocol}}{{mirror}}/web/simple some-package
```

</CodeBlock>

注意，`simple` 不能少，是 `https` 而不是 `http`

#### 设为默认

升级 pip 到最新的版本 (>=10.0.0) 后进行配置：

<CodeBlock>

```
python -m pip install --upgrade pip
pip config set global.index-url {{http_protocol}}{{mirror}}/web/simple
```

</CodeBlock>

如果您到 pip 默认源的网络连接较差，临时使用本镜像站来升级 pip：

<CodeBlock>

```
python -m pip install -i {{http_protocol}}{{mirror}}/web/simple --upgrade pip
```

</CodeBlock>

#### 配置多个镜像源

如果您想配置多个镜像源平衡负载，可在已经替换 `index-url` 的情况下通过以下方式继续增加源站：

```
pip config set global.extra-index-url "<url1> <url2>..."
```

请自行替换引号内的内容，源地址之间需要有空格

例如

<CodeBlock>

```
pip config set global.extra-index-url "{{http_protocol}}{{mirror}}/web/simple"
```

</CodeBlock>

### PDM

通过如下命令设置默认镜像：

<CodeBlock>

```
pdm config pypi.url {{http_protocol}}{{mirror}}/web/simple
```

</CodeBlock>

### Poetry

通过以下命令设置默认镜像：

<CodeBlock>

```
poetry source add --default mirrors {{http_protocol}}{{mirror}}/web/simple/
```

</CodeBlock>

通过以下命令设置次级镜像：

<CodeBlock>

```
poetry source add --secondary mirrors {{http_protocol}}{{mirror}}/web/simple/
```

</CodeBlock>
