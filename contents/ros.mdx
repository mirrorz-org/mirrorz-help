新建 `/etc/apt/sources.list.d/ros-latest.list`，内容为：

<CodeBlock
  enableQuickSetup
  filepath="/etc/apt/sources.list.d/ros-latest.list"
  menus={[
    {
      title: '发行版',
      items: [
        ['Ubuntu 20.04 LTS', { release_name: 'focal' }],
        ['Ubuntu 18.04 LTS', { release_name: 'bionic' }],
        ['Ubuntu 16.04 LTS', { release_name: 'xenial' }],
        ['Ubuntu 14.04 LTS', { release_name: 'trusty' }],
        ['Debian 10', { release_name: 'buster' }],
        ['Debian 9', { release_name: 'stretch' }],
        ['Debian 8', { release_name: 'jessie' }],
      ]
    }
  ]}
>


```properties
deb {{http_protocol}}{{mirror}}/ubuntu/ {{release_name}} main
```

</CodeBlock>

然后再输入如下命令，信任 ROS 的 GPG Key，并更新索引：

```bash
{{sudo}}apt-key adv --keyserver 'hkp://keyserver.ubuntu.com:80' --recv-key C1CF6E31E6BADE8868B172B4F42ED6FBAB17C654
{{sudo}}apt update
```
