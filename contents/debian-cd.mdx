Debian 安装镜像

需要安装 Debian 时，如果需要网络安装介质，下载以下链接

<CodeBlock>
```
{{http_protocol}}{{mirror}}/current/amd64/iso-cd/
```
</CodeBlock>

中以 ``amd64-netinst.iso`` 结尾的文件即可。

如果需要离线安装桌面环境，在

<CodeBlock>
```
{{http_protocol}}{{mirror}}/current-live/amd64/iso-hybrid/
```
</CodeBlock>

中可以选择不同桌面的 `debian-live-xxx-amd64-xxx.iso` 文件，如 `debian-live-12.1.0-amd64-lxqt.iso` 的文件中
就包含了离线安装 12.1.0 amd64架构 lxqt桌面的 Debian 所需的内容。
