---
title: Bazel APT 软件仓库镜像使用帮助
cname: 'bazel-apt'
---

使用以下命令启用 bazel-apt 仓库

<CodeBlock>

```bash
{{sudo}}apt install apt-transport-https curl gnupg
curl -fsSL https://bazel.build/bazel-release.pub.gpg | {{sudo}}gpg --dearmor -o /usr/share/keyrings/bazel-archive-keyring.gpg
echo "deb [arch=amd64 signed-by=/usr/share/keyrings/bazel-archive-keyring.gpg] {{http_protocol}}{{mirror}} stable jdk1.8" | {{sudo}}tee /etc/apt/sources.list.d/bazel.list
```

</CodeBlock>

使用以下命令安装 bazel

<CodeBlock>

```bash
{{sudo}}apt update && {{sudo}}apt install bazel
```

</CodeBlock>
