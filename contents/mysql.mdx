### Debian/Ubuntu 用户

再选择你的 Debian/Ubuntu 版本，文本框中内容写进 `/etc/apt/sources.list.d/mysql-community.list`

<CodeBlock
  enableQuickSetup
  filepath="/etc/apt/sources.list.d/mysql-community.list"
  menus={[
    {
      title: '发行版',
      items: [
        ['Debian 11', { os_name: 'debian', release_name: 'bullseye' }],
        ['Debian 10', { os_name: 'debian', release_name: 'buster' }],
        ['Debian 9', { os_name: 'debian', release_name: 'stretch' }],
        ['Debian 8', { os_name: 'debian', release_name: 'jessie' }],
        ['Ubuntu 22.04 LTS', { os_name: 'ubuntu', release_name: 'jammy' }],
        ['Ubuntu 20.04 LTS', { os_name: 'ubuntu', release_name: 'focal' }],
        ['Ubuntu 18.04 LTS', { os_name: 'ubuntu', release_name: 'bionic' }],
        ['Ubuntu 16.04 LTS', { os_name: 'ubuntu', release_name: 'xenial' }],
      ]
    },
    {
      title: 'MySQL 版本号',
      items: [
        ['8.0', { version: '8.0' }],
        ['5.7', { version: '5.7' }],
        ['5.6', { version: '5.6' }],
      ]
    }
  ]}
>

```properties
deb {{http_protocol}}{{mirror}}/apt/{{os_name}} {{release_name}} mysql-{{version}} mysql-tools
```

</CodeBlock>

参考文档：https://dev.mysql.com/doc/mysql-apt-repo-quick-guide/en/

### RHEL/CentOS 用户

新建 `/etc/yum.repos.d/mysql-community.repo`，内容如下：

注：`mysql-8.0`, `mysql-connectors`和`mysql-tools`在 RHEL 7/8上还提供了`aarch64`版本。

注：TUNA / BFSU 等镜像站不提供 CentOS 8 与 CentOS 6 版本


<CodeBlock
  enableQuickSetup
  filepath="/etc/yum.repos.d/mysql-community.repo"
  menus={[
    {
      title: '发行版',
      items: [
        ['CentOS/RHEL 7', { release_name: 'el7' }],
        ['CentOS/RHEL 8', { release_name: 'el8' }],
        ['CentOS/RHEL 6', { release_name: 'el6' }],
      ]
    },
    {
      title: 'MySQL 版本号',
      items: [
        ['8.0', { version: '8.0' }],
        ['5.7', { version: '5.7' }],
        ['5.6', { version: '5.6' }],
      ]
    }
  ]}
>

```ini
[mysql-connectors-community]
name=MySQL Connectors Community
baseurl={{http_protocol}}{{mirror}}/yum/mysql-connectors-community-{{release_name}}-$basearch/
enabled=1
gpgcheck=1
gpgkey=https://repo.mysql.com/RPM-GPG-KEY-mysql

[mysql-tools-community]
name=MySQL Tools Community
baseurl={{http_protocol}}{{mirror}}/yum/mysql-tools-community-{{release_name}}-$basearch/
enabled=1
gpgcheck=1
gpgkey=https://repo.mysql.com/RPM-GPG-KEY-mysql

[mysql-{{version}}-community]
name=MySQL {{version}} Community Server
baseurl={{http_protocol}}{{mirror}}/yum/mysql-{{version}}-community-{{release_name}}-$basearch/
enabled=1
gpgcheck=1
gpgkey=https://repo.mysql.com/RPM-GPG-KEY-mysql
```

</CodeBlock>

参考文档：https://dev.mysql.com/doc/mysql-yum-repo-quick-guide/en/
