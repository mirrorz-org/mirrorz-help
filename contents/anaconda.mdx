Anaconda 是一个用于科学计算的 Python 发行版，支持 Linux, Mac, Windows, 包含了众多流行的科学计算、数据分析的 Python 包。

Anaconda 安装包可以到以下链接下载。

<CodeBlock>

```
{{http_protocol}}{{mirror}}/archive/
```

</CodeBlock>

镜像站还提供了 Anaconda 仓库与第三方源（conda-forge、msys2、pytorch等，各镜像站镜像的第三方源并不相同）的镜像，各系统都可以通过修改用户目录下的 `.condarc` 文件来使用镜像站。Windows 用户无法直接创建名为 `.condarc` 的文件，可先执行 `conda config --set show_channel_urls yes` 生成该文件之后再修改。

注：由于更新过快难以同步，一些镜像站不同步`pytorch-nightly`, `pytorch-nightly-cpu`, `ignite-nightly`这三个包。

<CodeBlock>

```
channels:
  - defaults
show_channel_urls: true
default_channels:
  - {{http_protocol}}{{mirror}}/pkgs/main
  - {{http_protocol}}{{mirror}}/pkgs/r
  - {{http_protocol}}{{mirror}}/pkgs/msys2
custom_channels:
  conda-forge: {{http_protocol}}{{mirror}}/cloud
  msys2: {{http_protocol}}{{mirror}}/cloud
  bioconda: {{http_protocol}}{{mirror}}/cloud
  menpo: {{http_protocol}}{{mirror}}/cloud
  pytorch: {{http_protocol}}{{mirror}}/cloud
  pytorch-lts: {{http_protocol}}{{mirror}}/cloud
  simpleitk: {{http_protocol}}{{mirror}}/cloud
```

</CodeBlock>

即可添加 Anaconda Python 免费仓库。

运行 `conda clean -i` 清除索引缓存，保证用的是镜像站提供的索引。

运行 `conda create -n myenv numpy` 测试一下吧。
