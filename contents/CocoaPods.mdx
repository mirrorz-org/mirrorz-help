---
title: CocoaPods 软件仓库镜像使用帮助
cname: 'CocoaPods'
---

CocoaPods 是一个 Cocoa 和 Cocoa Touch 框架的依赖管理器，具体原理和 Homebrew 有点类似，都是从 GitHub 下载索引，然后根据索引下载依赖的源代码。

对于旧版的 CocoaPods 可以使用如下方法使用镜像：

<CodeBlock>

```bash
pod repo remove master
pod repo add master {{http_protocol}}{{mirror}}/Specs.git
pod repo update
```

</CodeBlock>

新版的 CocoaPods 不允许用`pod repo add`直接添加 master 库了，但是依然可以：

<CodeBlock>

```bash
$ cd ~/.cocoapods/repos 
$ pod repo remove master
$ git clone {{http_protocol}}{{mirror}}/Specs.git master
```

</CodeBlock>

最后进入自己的工程，在自己工程的`podFile`第一行加上：

<CodeBlock>

```
source '{{http_protocol}}{{mirror}}/Specs.git'
```

</CodeBlock>
