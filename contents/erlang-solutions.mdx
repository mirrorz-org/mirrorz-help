### Debian/Ubuntu 用户

请注意：官方提供的软件列表可能经常变化，镜像站不保证能提供所有系统与软件的组合。
请在[官网](https://www.erlang-solutions.com/downloads/)查询目前的可用版本，并根据实际情况指定版本。

首先信任 erlang-solutions 的 GPG 公钥：

<CodeBlock>

```bash
{{sudo}}mkdir -m 0755 -p /etc/apt/keyrings/
curl -s https://binaries2.erlang-solutions.com/GPG-KEY-pmanager.asc | {{sudo}}gpg --dearmor -o /etc/apt/keyrings/erlang-solutions-archive_compat.gpg
```

</CodeBlock>

新建 `/etc/apt/sources.list.d/erlang-solutions.list`，填入下列内容（可根据需要选择）：

<CodeBlock
  enableQuickSetup
  filepath="/etc/apt/sources.list.d/erlang-solutions.list"
  menus={[
    {
      title: '发行版',
      items: [
        ['Debian 12', { os_name: 'debian', release_name: 'bookworm' }],
        ['Debian 11', { os_name: 'debian', release_name: 'bullseye' }],
        ['Ubuntu 24.04 LTS', { os_name: 'ubuntu', release_name: 'noble' }],
        ['Ubuntu 22.04 LTS', { os_name: 'ubuntu', release_name: 'jammy' }],
      ]
    },
    {
      title: 'Mongoose IM 版本',
      items: [
        ['MongooseIM 5', { mongooseim_version: 'mongooseim-5' }],
        ['MongooseIM 4', { mongooseim_version: 'mongooseim-4' }],
      ]
    },
    {
      title: 'Erlang/OTP 版本',
      items: [
        ['Erlang/OTP 24', { erlang_version: 'esl-erlang-24' }],
        ['Erlang/OTP 25', { erlang_version: 'esl-erlang-25' }],
        ['Erlang/OTP 26', { erlang_version: 'esl-erlang-26' }],
      ]
    },
    {
      title: 'Elixir 版本',
      items: [
        ['Elixir 1.14', { elixir_version: 'elixir-1.14' }],
        ['Elixir 1.15', { elixir_version: 'elixir-1.15' }],
      ]
    }
  ]}
>

```properties
deb [signed-by=/etc/apt/keyrings/erlang-solutions-archive_compat.gpg] {{http_protocol}}{{mirror}}/{{os_name}} {{release_name}}-esl-erlang-{{erlang_version}} contrib
deb [signed-by=/etc/apt/keyrings/erlang-solutions-archive_compat.gpg] {{http_protocol}}{{mirror}}/{{os_name}} {{release_name}}-elixir-{{elixir_version}} contrib
deb [signed-by=/etc/apt/keyrings/erlang-solutions-archive_compat.gpg] {{http_protocol}}{{mirror}}/{{os_name}} {{release_name}}-mongooseim-{{mongooseim_version}} contrib
```

</CodeBlock>


安装 `esl-erlang` 等软件包即可：

<CodeBlock>
```bash
{{sudo}}apt-get update
{{sudo}}apt-get install -y elixir esl-erlang mongooseim
```
</CodeBlock>
