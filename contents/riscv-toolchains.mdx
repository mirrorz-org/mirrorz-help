本镜像包含 RISC-V 常用工具与文档，分别以 Git 仓库与 Release 的形式镜像。

### Git 仓库

本站提供了克隆脚本，以供克隆相关仓库与子模块。样例如下

<CodeBlock>

```bash
# riscv-gnu-toolchain 及其子模块
## 单独克隆
git clone {{http_protocol}}{{mirror}}/git/riscv-collab/riscv-gnu-toolchain.git
## 同时克隆子模块，注意路径为 .git 替换为 .sh
curl -L {{http_protocol}}{{mirror}}/git/riscv-collab/riscv-gnu-toolchain.sh | bash
# rocket-tools 及其子模块
## 单独克隆
git {{http_protocol}}{{mirror}}/git/chipsalliance/rocket-tools.git
## 同时克隆子模块
curl -L {{http_protocol}}{{mirror}}/git/chipsalliance/rocket-tools.sh | bash
```

</CodeBlock>

已有项目可以在

<CodeBlock>

```
{{http_protocol}}{{mirror}}/git/
```

</CodeBlock>

中浏览。

### Release

镜像站提供了标准文档和预编译工具链的镜像，例如


<CodeBlock>

```bash
# RISC-V 非特权与特权指令集文档
{{http_protocol}}{{mirror}}/release/riscv/riscv-isa-manual/
# RISC-V 向量扩展指令集文档
{{http_protocol}}{{mirror}}/release/riscv/riscv-v-spec/
# 预编译 riscv-gnu-toolchain
{{http_protocol}}{{mirror}}/release/riscv-collab/riscv-gnu-toolchain/LatestRelease/
```

</CodeBlock>

已有项目可以在

<CodeBlock>

```
{{http_protocol}}{{mirror}}/release/
```

</CodeBlock>

中浏览。
