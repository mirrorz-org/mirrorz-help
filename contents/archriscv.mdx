### 软件源

编辑 `/etc/pacman.d/mirrorlist`，在文件的最顶端添加：

<CodeBlock
  enableQuickSetup
  filepath="/etc/pacman.d/mirrorlist"
>

```ini
Server = {{http_protocol}}{{mirror}}/repo/$repo
```

</CodeBlock>

其余部分可以参考 [Arch Linux 帮助](/archlinux/)。

### 系统镜像

提供的镜像为 rootfs，可在目标目录中通过 `tar xf` 解压后使用。

<CodeBlock>
```
{{http_protocol}}{{mirror}}/images/
```
</CodeBlock>
