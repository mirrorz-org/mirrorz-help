cname: 'rudder'
---

### Debian/Ubuntu 用户


选择你的 Debian/Ubuntu 版本，文本框中内容写进 `/etc/apt/sources.list.d/rudder.list`

<CodeBlock
  enableQuickSetup
  filepath="/etc/apt/sources.list.d/rudder.list"
  menus={[
    {
      title: '发行版',
      items: [
        ['Debian 13', { release_name: 'trixie' }],
        ['Debian 12', { release_name: 'bookworm' }],
        ['Debian 11', { release_name: 'bullseye' }],
        ['Ubuntu 24.04 LTS', { release_name: 'noble' }],
        ['Ubuntu 22.04 LTS', { release_name: 'jammy' }],
      ]
    },
    {
      title: 'Rudder 版本号',
      items: [
        ['最新版', { version: 'latest' }],
      ]
    }
  ]}
>

```properties
deb {{http_protocol}}{{mirror}}/apt/{{version}} {{release_name}} main
```

</CodeBlock>

### RHEL/CentOS 用户

新建 `/etc/yum.repos.d/rudder.repo`，内容为：

<CodeBlock
  enableQuickSetup
  filepath="/etc/yum.repos.d/rudder.repo"
  menus={[
    {
      title: 'Rudder 版本号',
      items: [
        ['最新版', { version: 'latest' }],
      ]
    }
  ]}
>

```ini
[Rudder_{{version}}]
name=Rudder {{version}}
baseurl={{http_protocol}}{{mirror}}/rpm/rudder{{version}}-RHEL_$releasever/
enabled=1
gpgcheck=1
gpgkey=https://repository.rudder.io/rpm/rudder_rpm_key.pub
```

</CodeBlock>
