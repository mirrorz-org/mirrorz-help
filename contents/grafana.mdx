---
title: Grafana 软件仓库镜像使用帮助
cname: 'grafana'
---
### Debian / Ubuntu 用户

首先信任 https://apt.grafana.com/ 的 GPG 公钥：

<CodeBlock>
```bash
{{sudo}}wget -q -O /etc/apt/keyrings/grafana.key https://apt.grafana.com/gpg.key
```
</CodeBlock>

确保你的 apt 支持 HTTPS:

<CodeBlock>
```bash
{{sudo}}apt-get install -y apt-transport-https
```
</CodeBlock>

选择你希望安装的 Grafana 版本（与你的 Debian/Ubuntu 系统版本无关），文本框中内容写进 `/etc/apt/sources.list.d/grafana.list`

<CodeBlock
  enableQuickSetup
  filepath="/etc/apt/sources.list.d/grafana.list"
  menus={[
    {
      title: 'Grafana 版本',
      items: [
        ['Stable', { version: 'stable' }],
        ['Beta', { version: 'beta' }],
      ]
    }
  ]}
>

```properties
deb [signed-by=/etc/apt/keyrings/grafana.key] {{http_protocol}}{{mirror}}/apt/ {{version}} main
```

</CodeBlock>


安装 Grafana

<CodeBlock>
```shell
{{sudo}}apt-get update
{{sudo}}apt-get install grafana
```
</CodeBlock>

### Centos / Redhat 用户

新建 `/etc/yum.repos.d/grafana.repo`，内容为

<CodeBlock
  enableQuickSetup
  filepath="/etc/yum.repos.d/grafana.repo"
>

```ini
[grafana]
name=grafana
baseurl={{http_protocol}}{{mirror}}/yum/rpm
repo_gpgcheck=0
enabled=1
gpgcheck=0
```

</CodeBlock>

如果要避免安装 beta 版本，需要新增一行

```ini
exclude=*beta*
```

再执行

<CodeBlock>
```shell
{{sudo}}yum makecache
{{sudo}}yum install grafana
```
</CodeBlock>
