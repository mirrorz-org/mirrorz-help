```bash
# 下载脚本
wget {{http_protocol}}{{mirror}}/llvm.sh
chmod +x llvm.sh
{{sudo}}./llvm.sh {{version}} all -m {{http_protocol}}{{mirror}}
```

</CodeBlock>

### 手动启用

首先信任来自 https://apt.llvm.org/ 的 PGP 公钥：

<CodeBlock>
```bash
wget -O - https://apt.llvm.org/llvm-snapshot.gpg.key | {{sudo}}apt-key add -
```
</CodeBlock>

新增 `/etc/apt/sources.list.d/llvm-apt.list`，内容为

<CodeBlock
  enableQuickSetup
  filepath="/etc/apt/sources.list.d/llvm-apt.list"
  menus={[
    {
      title: '发行版',
      items: [
        ['Debian 11', { release_name: 'bullseye' }],
        ['Debian 10', { release_name: 'buster' }],
        ['Ubuntu 22.04 LTS', { release_name: 'jammy' }],
        ['Ubuntu 20.04 LTS', { release_name: 'focal' }],
        ['Ubuntu 18.04 LTS', { release_name: 'bionic' }],
      ]
    },
    {
      title: 'LLVM 版本',
      items: [
        ['默认', { version: '' }],
        ['16', { version: '-16' }],
        ['15', { version: '-15' }],
        ['14', { version: '-14' }],
        ['13', { version: '-13' }],
        ['12', { version: '-12' }],
        ['11', { version: '-11' }],
      ]
    },
    {
      title: '启用源码镜像',
      items: [
        ['否', { enable_source: '# ' }],
        ['是', { enable_source: '' }],
      ]
    }
  ]}
>

```properties
# 默认注释了源码镜像以提高 apt update 速度，如有需要可自行取消注释
deb {{http_protocol}}{{mirror}}/{{release_name}}/ llvm-toolchain-{{release_name}}{{version}} main
{{enable_source}}deb-src {{http_protocol}}{{mirror}}/{{release_name}}/ llvm-toolchain-{{release_name}}{{version}} main
```

</CodeBlock>
